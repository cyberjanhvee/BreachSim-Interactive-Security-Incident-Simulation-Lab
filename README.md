BreachSim is an interactive cybersecurity simulation dashboard designed to model common attack scenarios in a controlled, educational environment.
It simulates phishing campaigns, malware execution in a sandbox, and brute-force attacks while generating real-time logs and exportable JSON incident reports.

*Features
>Phishing campaign simulation with click & compromise tracking
>Malware sandbox behavior emulation (ransomware, spyware, RAT)
>Brute-force attack simulation with firewall response
>Real-time terminal-style incident logs
>JSON export of generated incident data

*Tech Stack
>HTML
>CSS
>JavaScript
